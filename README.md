#Penspinning Academy

A production web application built with Next.js 15, React, TypeScript and Tailwind CSS — actively used by a community of 2,000+ users.

##Technical Highlights

Data-driven architecture — trick content is generated dynamically from a structured data layer, keeping UI components fully decoupled from content. Adding new content requires zero UI changes.
Next.js App Router — leverages server components, dynamic routing ([level]/[trick]), and static generation for fast page loads.
Persistent user preferences — dark mode and user progress stored via localStorage, synced on mount without layout shift.
Scalable classification system — each trick is typed and tagged by family, difficulty and modifiers, enabling filtered views and future search features.
Fully responsive — mobile-first layout with Tailwind, tested across screen sizes.
Deployed on Vercel — CI/CD on push to main, zero-downtime deployments.
